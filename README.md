- @ = At Sign
- \# = Number Sign
- $ = Dollar Sign
- % = Percentage Sign
- ^ = Circumflex
- & = Ampersand
- \* = Asterisk
- ( = Parenthesis (Left)
- ) = Parenthesis (Right)
